# chat_server

Chat para multiplos usuários em tempo real com as funcionalidades:
 
  - Auntenticação
  - Criação de sala
  - Edição e exclusão de sala pelo dono
  - Chamar usuário da conversa no privado
  
A aplicação pode ser acessada em http://134.209.223.242:3000

## Tecnologias

  - NodeJS, Express
  - MySQL 
  - Docker
  - Swagger
  - Socket.io
